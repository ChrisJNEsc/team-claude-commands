# Session Data

This directory contains session logs from /analyze-bug command runs.

## Purpose
- Document investigation sessions for team learning
- Track patterns and improve bug analysis processes
- Share context between team members

## File Naming Convention
`[YYYY-MM-DD]-[LINEAR-ID].md` or `[YYYY-MM-DD]-[UNKNOWN]-[topic].md`

## Usage
Run `/save-session` after completing an `/analyze-bug` session to save the context here.
